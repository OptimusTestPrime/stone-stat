How to Use StoneStat

Getting Started

StoneStat tracks shot-by-shot performance for two teams of four players. Before starting a game, fill in the Game Setup screen:

•Number of Ends - Choose 4, 6, 8 or 10 ends.
•Hammer (End 1) - Select which team has hammer in the first end.
•Team Names and Players - Enter names for both teams and all four players (Lead, Second, Third, Skip).

Tap Start Game when ready.

Recording a Shot

The app follows WCF shot order automatically - non-hammer team throws first, alternating between teams. For each shot, fill in the Record Shot panel:

•Handle / Rotation - Select In-Turn or Out-Turn.
•Shot Type - Choose from Draw, Guard, Hit, Hit and Roll, Freeze, or Raise.
•Miss Type - Optionally select Heavy, Light, Wide, or Tight. Heavy/Light are mutually exclusive, as are Wide/Tight.
•Score - Rate the shot 0 to 4, or X for a deliberate throwaway.
•Notes - Add an optional note about the shot.

Tap Record Shot to save. Handle and Shot Type are mandatory. Use the Skip button (top right) to pass on a shot.

Score Scale
•4Perfect - Completely achieved the intended result.
•3Good - Mostly achieved, minor error.
•2Fair - Partially achieved, notable error.
•1Poor - Small contribution, significant error.
•0Missed - Did not execute, no positive contribution.
•XThrowaway - Deliberate throw-through. Excluded from % calculations.
Navigation
•Prev Shot - Go back to the previous shot to review or edit it.
•Skip to End Score - Jump directly to the end score entry.
•End Score Entry - After all 16 shots, enter how many points each team scored. Only one team scores per end.
•Scoreboard Ends - Tap any end number or score cell in the scoreboard to jump directly to that end.
Statistics

Player panels update in real time. Each player row shows shot dots coloured by score and per-type percentages broken down by handle - e.g. D In:72% Out:68%.

Tap Stats in the header to see a full game summary including team averages and per-player breakdown by shot type and handle.

Shot Type Key
D = DrawG = GuardH = HitHR = Hit and RollFr = FreezeR = Raise
Game History

The last 20 completed games are saved automatically in your browser. Tap History to view past games and export data. Each game can be exported as:

•CSV - Open in Excel or Google Sheets.
•JSON - Raw data for custom analysis.
•Text - Plain text game report.

On iPhone, the iOS share sheet opens so you can save to Files, AirDrop, or any app.

Tips
•Add StoneStat to your iPhone home screen via Safari for a full-screen app experience (tap Share then "Add to Home Screen").
•The app works fully offline once loaded.
•Game history is stored in your browser - clearing browser data will remove saved games.
•Hammer rotates automatically based on who scores each end.
