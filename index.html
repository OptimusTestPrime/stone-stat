How to Use StoneStat

Getting Started

StoneStat tracks shot-by-shot performance for two teams. Before starting, fill in the Game Setup screen:

•Number of Ends - Choose 4, 6, 8 or 10 ends.
•Hammer (End 1) - Select which team has hammer in the first end.
•Team Names and Players - Enter names for both teams and all four players (Lead, Second, Third, Skip).
•3-Player Team - Check this box under a team if they are playing with three players (no Third). The shot distribution becomes Lead x3, Second x3, Skip x2.
•Fields to Record - Choose which fields appear in the Record Shot panel. Score is always required. Handle/Rotation, Shot Type, Miss Type, and Notes can each be turned off for simpler scoring.

Tap Start Game when ready.

Recording a Shot

The app follows WCF shot order automatically - non-hammer team throws first, alternating between teams. For each shot, fill in the Record Shot panel:

•Handle / Rotation - Select In-Turn or Out-Turn.
•Shot Type - Choose from Draw, Guard, Hit, Hit and Roll, Freeze, or Raise.
•Miss Type - Optionally select Heavy, Light, Wide, or Tight (or a combination). Heavy/Light are mutually exclusive, as are Wide/Tight.
•Score - Rate the shot 0 to 4, or X (Throwaway) for a deliberate throw-through.
•Notes - Add an optional note about the shot.

Tap Record Shot to save and advance. Use the Skip button (top right of the panel) to pass on recording a shot without entering any data.

Tap any player row in the Red or Yellow team panels to override which player the shot is recorded for. Tap the active player again to return to automatic shot order.

Score Scale
•4Perfect - Completely achieved the intended result.
•3Good - Mostly achieved, minor error.
•2Fair - Partially achieved, notable error.
•1Poor - Small contribution, significant error.
•0Missed - Did not execute, no positive contribution.
•XThrowaway - Deliberate throw-through. Excluded from percentage calculations.
Navigation
•Prev Shot - Go back to the previous shot to review or edit it. Score, handle, shot type, miss type, and notes are all restored for editing.
•Skip - Pass on recording a shot. Shown as a dashed dot in the shot tracker.
•Skip to End Score - Jump directly to end score entry at any time.
•End Score Entry - After all 16 shots, enter how many points each team scored. Only one team scores per end. Hammer rotates automatically to the team that did not score.
•Scoreboard Ends - Tap any end number or score cell in the scoreboard to jump directly to that end.
Statistics

Player panels update in real time. Each player row shows:

•Shot dots coloured by score (red = missed, yellow = fair, teal = perfect).
•Per-type percentages broken down by handle - e.g. D In:72% Out:68%. Only types and handles that have been recorded appear.

Tap Stats in the header for a full game summary including team averages, per-player type/handle breakdown, and a legend explaining all abbreviations.

Shot Type and Abbreviation Key
D = DrawG = GuardH = HitHR = Hit and RollFr = FreezeR = Raise

In the stats panels: In = In-Turn, Out = Out-Turn. Miss types: Heavy, Light, Wide, Tight.

Game History and Export

The last 20 completed games are saved automatically in your browser. Tap History to view past games. Each game can be exported in three formats:

•CSV - Spreadsheet format including end scores, full shot log (End, Shot, Team, Player, Position, Turn, Shot Type, Miss Type, Score, Label, Note), and player stats. Open in Excel or Google Sheets.
•JSON - Complete raw game data for custom analysis.
•Text - Plain text game report with shot log showing all recorded details per shot.

On iPhone, tapping an export button opens the iOS share sheet so you can save to Files, AirDrop, Notes, or any compatible app.

Tips
•Add StoneStat to your iPhone home screen via Safari for a full-screen app experience - tap the Share icon then "Add to Home Screen".
•The app works fully offline once loaded in your browser.
•Game history is stored in your browser's local storage. Clearing browser data will remove saved games.
•Hammer rotates automatically: the team that did not score gets hammer next end. If neither team scores, hammer stays.
•For quick scoring sessions, uncheck Handle/Rotation, Shot Type, Miss Type, and Notes in Game Setup - then just tap a score for each shot.
•Use the scoreboard end numbers to quickly jump between ends to review or correct shots.
